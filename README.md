# 日圓走勢分析工具

一個基於 Vue 3 開發的匯率分析應用程式，專門用於追蹤與分析日圓（JPY）與新台幣（TWD）之間的匯率走勢變化。

## ✨ 功能特色

### 📊 匯率圖表分析
- **雙向匯率查詢**：支援 TWD/JPY 與 JPY/TWD 雙向匯率查詢
- **時間區間選擇**：
  - 本日
  - 三日內
  - 一週
  - 一個月
  - 自訂日期範圍
- **移動平均線**：自動計算並顯示 7 日與 30 日移動平均線（MA7、MA30）
- **統計標記**：圖表標示最高點、最低點與平均值

### 🔄 多幣別對照
- 支援與其他幣別對照分析：
  - USD/JPY（美元對日圓）
  - EUR/JPY（歐元對日圓）
  - CNY/JPY（人民幣對日圓）
- 次軸顯示功能，讓不同比例的匯率可以同時清楚對照

### ⚡ 即時更新
- **自動更新**：在查看當日匯率時可啟用自動更新功能
- **更新頻率可調**：支援 1 分鐘、5 分鐘、15 分鐘等更新間隔

### 📥 資料匯出
- **CSV 匯出**：匯出匯率資料、移動平均線及對照幣別資料
- **PNG 圖表匯出**：儲存高解析度圖表影像

### 💡 即時統計
- 顯示當前匯率
- 計算區間內匯率變化（絕對值與百分比）

## 🛠️ 技術架構

### 前端框架
- **Vue 3**：採用 Composition API 開發
- **Vite**：快速的建置工具與開發伺服器

### UI 框架
- **Bootstrap 5**：響應式設計與元件

### 圖表庫
- **Apache ECharts**：專業的數據視覺化圖表庫

### 工具函式庫
- **Day.js**：輕量級的日期處理工具

### 資料來源
- **fawazahmed0 Currency API**：免費的匯率 API，支援 CORS

## 🔧 主要元件說明

### [`App.vue`](src/App.vue)
主要應用程式元件，包含：
- 時間範圍選擇器
- 幣別切換
- 對照幣別選擇
- 自動更新控制
- 資料匯出功能
- 匯率統計資訊顯示

### [`RateChart.vue`](src/components/RateChart.vue)
圖表顯示元件，使用 ECharts 繪製：
- 支援主要匯率與對照匯率的雙軸顯示
- 移動平均線疊加
- 互動式 tooltip
- 圖表匯出功能

### [`rates.js`](src/services/rates.js)
匯率資料服務：
- 封裝 API 呼叫邏輯
- 處理日期範圍查詢
- 支援多幣別查詢
- 錯誤處理與重試機制

## 📝 使用說明

1. **選擇時間範圍**：使用預設時間按鈕或自訂日期範圍
2. **切換幣別**：選擇 TWD/JPY 或 JPY/TWD
3. **新增對照**：從下拉選單選擇要對照的幣別
4. **啟用次軸**：當對照幣別的匯率比例差異較大時，啟用次軸可以更清楚比較
5. **自動更新**：查看當日匯率時可啟用自動更新，並設定更新頻率
6. **匯出資料**：使用「匯出 CSV」或「匯出 PNG」按鈕儲存資料
